# Faceify
A publicly sourced skin-care database and social platform!
Designed using a mobile first approach


![landing](faceify/../public/images/landing2.png)

# Technologies Used:
- Node.js
- Express
- MongoDB Atlas
- Mongoose
- EJS Templating
- Google OAuth
- Github
- Heroku
- Visual Code Studio

# Getting Started:
