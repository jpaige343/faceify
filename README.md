# Faceify
A publicly sourced skin-care database!


![landing](personalized-shopping/../public/images/landing.png)

# Technologies Used:
- Node.js
- Express
- MongoDB Atlas
- Mongoose
- EJS Templating
- Google OAuth
- Github
- Heroku
