# Faceify
A publicly sourced skin-care database and social platform!
Designed with a mobile first approach


![landing](personalized-shopping/../public/images/landing.png)

# Technologies Used:
- Node.js
- Express
- MongoDB Atlas
- Mongoose
- EJS Templating
- Google OAuth
- Github
- Heroku
- Visual Code Studio
